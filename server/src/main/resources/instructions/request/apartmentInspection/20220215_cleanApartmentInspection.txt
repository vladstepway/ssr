Затереть организации в акте по дефектам (id = 2186dfd8-21e5-4dee-a61c-fa634c82ecbb) и перепривязать задачу на МФР (сняв её с человека).

Выполнение инструкции возможно только в Postman версии не ниже чем v.7.9.0

1) Загрузить в Postman через File->Import коллекцию 20220215_cleanApartmentInspection.postman_collection.json

2) Открыть запрос "1) fetch document", зайти на вкладку "Pre-request Script", добавить актуальное значение Cookie.
Для этого необходимо войти в систему под учетной записью пользователя. Перейти в Developer Tool (нажать F12). Выбрать закладку Network, перейти на XHR/Fetch. Обновить страницу. Нажать на любое значение из крайнего левого столбца в появившемся списке. Перейти на закладку Headers. Найти Request, внутри которого будет элемент Cookie. Скопировать его значение (то, что будет после "Cookie: "). Скопированное значение необходимо вставить внутрь кавычек с пустой строкой внутри в pm.collectionVariables.set("cookie", "")

3) Выбрать загруженную коллекцию и нажать кнопку Run (кнопка расположена в главном окне возле наименования коллекции справа сверху)

4) В открывшемся окне нажать синюю кнопку Run cleanApartmentInspection

5) Убедиться, что все статусы выполненных запросов 200 или 204, а также проверить консоль на предмет ошибок и выполненных действий (кнопку Console можно найти в левом нижнем углу)
