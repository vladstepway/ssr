Загрузка данных о письмах с предложениями.

1. Открыть https://smart.mos.ru/app/ugd/ssr/swagger-ui.html#/person-controller
2. Выбрать POST /person/upload-offer-letter
3. Нажать Try it out
4. Загрузить файл "Файл для загрузки - Выгрузка из К3 для сверки с ДГП.xlsx" в file, нажав Choose File и выбрав файл
5. Нажать Execute и убедиться, что статус код ответа 200
